== Description ==

Child theme build for Understrap, a renowned WordPress starter theme framework that combines Underscores and Bootstrap.

Website: https://www.isopods.ca

Child Theme Project:

== For Devs ==
1. Do npm install from src directory
2. Do npm run watch to compile SASS and JS and watch for changes
3. Do npm run build to build production files

== For End Users ==
1. Copy both the Understrap them and this child theme to the themes directory
2. Activate this child theme in admin dashboard
3. Install ACF Pro and WooCommerce plugins
